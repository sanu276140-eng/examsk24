# ExamsK24 Admin Panel

Admin panel for managing examsk24.online website.

## Setup Instructions

1. Clone the repository
2. Create Firebase project
3. Update Firebase config in admin.html
4. Create first admin user using create-admin.js
5. Deploy to hosting

## File Structure

- admin.html - Main admin panel
- admin-script.js - Admin JavaScript
- style.css - Admin styles
- create-admin.js - First admin setup script
